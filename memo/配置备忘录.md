# latex公式字体加粗
下面总结常用的几种加粗方法，并提供一个精简的表格方便查阅。

名称	    用途	    支持希腊字母
\mathbf	拉丁字母、数字	    否
\boldsymbol	希腊字母、符号	是
\bm  	最强加粗	       是
\pmb	伪粗体	          是
1、\mathbf{}：最适用于拉丁字母和数字，但不能加粗希腊字母。

2、\boldsymbol{}（需要 amsmath）：可加粗希腊字母、符号，是数学环境中最常用的方案之一。

3、\bm{}（bm 宏包）：更强大的加粗方式，基本可以加粗所有数学符号。

4、\pmb{}：通过错位显示实现“伪粗体”，不依赖宏包，但效果较差。
